Kifu for JS [v5.0.3](https://github.com/na2hiro/Kifu-for-JS/releases/tag/v5.0.3) (と[v5.0.2](https://github.com/na2hiro/Kifu-for-JS/releases/tag/v5.0.2)) パッチリリースにおいて、以下のバグ修正を行いました。

- Safariでの表示乱れと再生時にクラッシュするバグを修正

- その他表示の微調整

v5リリース直後であるため、まだまだバグが見つかるかもしれません。おかしなところがありましたら [GitHub issues](https://github.com/na2hiro/Kifu-for-JS/issues) または [@na2hiro](https://twitter.com/na2hiro) までご報告ください。よろしくお願いします。
