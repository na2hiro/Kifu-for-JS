# 局面の指定

最初に表示する局面を指定できます。

<TOCInline toc={toc} />

## X手目の局面から

X手目の局面から表示する場合、`data-ply` 属性を指定します。

例えば、12手目の局面を表示する場合：

```html
<script type="text/kifu" data-ply="12">
▲７六歩　△３四歩…
</script>
```

* デフォルトは0で、棋譜の一番最初の局面を表示します。
* 999などの大きな数を指定すると、最後の局面を表示します。

## 分岐局面から

分岐局面から表示する場合、どの分岐を辿ったかを表す `data-forkpointers` 属性を指定します。

例えば、12手目の局面だが、5手目の局面での2番目の分岐を選び、その後11手目の局面で更に1番目の分岐を選んだ局面の場合：

```html
<script type="text/kifu" data-ply="12" data-forkpointers="[[5, 2], [11, 1]]">
▲７六歩　△３四歩…
</script>
```

* 本譜を進む場合は、`forkpointers` に現れません。
* 注意：分岐の指定は、1から始まることに注意してください。
