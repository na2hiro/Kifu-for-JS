# 静的図面モード (`static`)

再生盤ではなく、動かすことのできない静的な局面画像を表示することができます。盤面領域のみが横6:縦5 (300:250)で表示されます。棋譜解説などの際、一つずつ画像を生成する手間を省けます。

これには `static` オプションを指定します。

## 棋譜中の局面を指定する場合

<KifuForEachMethod static ply={3}>
▲７六歩 △３四歩 ▲７五歩 △８四歩
</KifuForEachMethod>

[局面の指定](./specifying-ply)も参照してください。

### 着手表示を抑制する

通常、初期局面や局面読み込みなどの場合を除き、最終手が着手表示され太字になります。 `static-last` オプションに `hidden` を指定すると、着手表示を抑制することができます。

<KifuForEachMethod ply={3} static={{last:"hidden"}}>
  ▲７六歩 △３四歩 ▲７五歩 △８四歩
</KifuForEachMethod>

SFEN形式にも対応しています。

<KifuForEachMethod static>
ln1gkg1nl/6+P2/2sppps1p/2p3p2/p8/P1P1P3P/2NP1PP2/3s1KSR1/L1+b2G1NL w R2Pbgp 42
</KifuForEachMethod>

### 着手表示箇所を指定する

局面データには着手というものが存在しないため、着手表示をするには着手したマスを指定します。

３二へ着手表示をする場合、`static-last` オプションに `[3, 2]` を指定します。

<KifuForEachMethod static={{last: [3, 2]}}>
ln1gkg1nl/6+P2/2sppps1p/2p3p2/p8/P1P1P3P/2NP1PP2/3s1KSR1/L1+b2G1NL w R2Pbgp 42
</KifuForEachMethod>
