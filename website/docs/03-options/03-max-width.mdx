# 最大横幅 (`maxWidth`)

詳しくは[レイアウトを調整する#モバイル/デスクトップ両対応レイアウト](../adjusting-layout#%E3%83%A2%E3%83%90%E3%82%A4%E3%83%AB%E3%83%87%E3%82%B9%E3%82%AF%E3%83%88%E3%83%83%E3%83%97%E4%B8%A1%E5%AF%BE%E5%BF%9C%E3%83%AC%E3%82%A4%E3%82%A2%E3%82%A6%E3%83%88)をご覧ください。

## 最大横幅のデフォルトは400px

<KifuExample>
    ▲７六歩 △３四歩 ▲７五歩 △８四歩
</KifuExample>

## 最大横幅を指定し直す

<KifuExample maxWidth={300}>
    ▲７六歩 △３四歩 ▲７五歩 △８四歩
</KifuExample>

## 最大横幅を制限しない

<KifuExample maxWidth={null}>
    ▲７六歩 △３四歩 ▲７五歩 △８四歩
</KifuExample>
